Node Output Panel

When you click a node, its output appears in the right panel. How should it be displayed?

[ analyze ] OUTPUT ✕ 
{
  "analysis": "Strategic deterrence signals are elevated...",
  "confidence": 0.87,
  "frameworks": [
    "realism",
    "game-theory"
  ],
  "key_risks": [
    "Escalation ladder miscalculation",
    "Alliance cohesion under stress"
  ],
  "recommendation": "Monitor..."
}

A — Pretty-printed JSON

Syntax-highlighted, indented JSON. Always accurate, works for any node regardless of output shape. Good for developers who want to see raw data.

[ analyze ] OUTPUT ✕
analysis
Strategic deterrence signals are elevated. Alliance posture suggests...
confidence
0.87
frameworks
realism game-theory
key risks
• Escalation ladder miscalculation
• Alliance cohesion under stress

B — Structured key/value display

Parses the JSON and renders each top-level key as a labeled section. Strings as text, numbers highlighted, arrays as tags or bullets. More readable, but requires output to be a flat-ish object.

[ analyze ] OUTPUT
FORMATTED RAW ✕
analysis
Strategic deterrence signals are elevated...
confidence
0.87
frameworks
realism game-theory 
// RAW tab would show full JSON here

C — Formatted view + Raw tab toggle

Defaults to the structured view (like B), but a "RAW" tab lets you switch to the full JSON. Best of both worlds — readable by default, inspectable when needed.

Click your preference, then let me know in the terminal.
